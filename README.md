# IBRUSH - Тестовое задание для верстальщиков
## Установка

Необходимо иметь [Node.js](https://nodejs.org/) для запуска и [Yarn](https://yarnpkg.com/) . 

Установить все зависимости командой:
```sh
yarn
```

Для начала разработки:
```sh
gulp
```

Чтобы собрать проект

```sh
gulp build
```
